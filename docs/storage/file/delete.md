# itp storage file delete

Delete a file from an iTwin's storage.

## Options

- **`--fileId`**  
  The ID of the file to be deleted.  
  **Type:** `string` **Required:** Yes

## Examples

```bash
itp storage file delete --fileId "bf4d8b36-25d7-4b72-b38b-12c1f0325f42"
```
