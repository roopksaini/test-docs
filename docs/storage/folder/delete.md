# itp storage folder delete

Delete a folder from an iTwin's storage.

## Options

- **`--folderId`**  
  The ID of the folder to be deleted.  
  **Type:** `string` **Required:** Yes

## Examples

```bash
itp storage folder delete --folderId "a1b2c3d4-5678-90ab-cdef-1234567890ab"
```
