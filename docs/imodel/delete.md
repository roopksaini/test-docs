# itp imodel delete

Delete an existing iModel.

## Options

- **`--id`**  
  The ID of the iModel to delete.  
  **Type:** `string` **Required:** Yes

## Examples

```bash
itp imodel delete --id "5e19bee0-3aea-4355-a9f0-c6df9989ee7d"
```
