# itp itwin delete

Delete the specified iTwin.

## Options

- **`--id`**  
  The ID of the iTwin to be deleted.  
  **Type:** `string` **Required:** Yes

## Examples

```bash
itp itwin delete --id "b1a2c3d4-5678-90ab-cdef-1234567890ab"
```
